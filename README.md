A simple ConsoleCalculator that models basic arithemetic calculations. 
User's can  perform either Addition, Subtraction, Multiplication and Division.

User's perform these calculations by first entering a "First" operand, followed by the desired operator (+, -, /, *) and then a "Second" operand.
User's can choose to continue calculating by entering the desired operator after each sucessive calculations or press the "End" key to exit application.
